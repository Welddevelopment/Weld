17 · Profile Live! 🎉
↑ After publish → drops into matching pool. Studios start swiping immediately.
🔑 Auth methods
OAuth (Roblox, Google, Discord) → skips email confirm step, auto-fills identity fields
Email+password → requires confirm email step before entering builder
Draft saves locally on every change; syncs to account once signed in
🔗 Social linking notes
Roblox link: required for trust badge + avatar pull-in (can use URL fallback)
Discord: shown prominently on card — studios DM devs there
GitHub, X, YouTube: optional, each adds to trust score
Can add/remove socials later from profile settings
🎭 Two paths (Dev vs Studio)
Screens 11–15 shown above are Dev path
Studio path replaces Rate with Team Size + Status + Budget
Studio Work step = Top Games (3 max) instead of Projects
Studio has no Portfolio links step (5 steps vs 6)
💾 Data persistence
LocalStorage draft saves on every change
Account sync saves to Supabase when logged in
User can exit mid-flow and resume from same step
✅ Step summary (Dev)
0 Type: Dev or Studio
1 Identity: name, Roblox link, socials, colour, availability
2 Role: experience + rate type + amount
3 Bio: short pitch + details
4 Skills: up to 5, with descriptions
5 Work: up to 3 projects + stats
6 Portfolio: links, then preview + publish
