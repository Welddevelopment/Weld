# Claude: Build the Weld App

This document is a guide for building the Weld application. It includes the core frontend design skill and other app-building skills that should be used globally across the project.

## Weld App Overview

Weld is a product-focused app built for modern web experiences. The main goal is to create a polished, responsive, and easy-to-use frontend while supporting reliable backend interactions.

## Frontend Design Skill

Use the following frontend design principles consistently:

- User-centered UI design
- Responsive layouts for desktop and mobile
- Clear visual hierarchy and spacing
- Consistent branding and typography
- Intuitive navigation flows
- Accessible components and semantic HTML
- Polished interactions and animation where appropriate

## Global App Building Skills

These skills should be applied globally across the Weld application and all related work:

- Product design thinking
- UI/UX strategy
- Component-based frontend architecture
- Responsive web design
- Accessibility best practices (WCAG)
- Performance optimization
- API integration and data handling
- Cross-browser compatibility
- Testing and validation
- Collaboration with designers, developers, and stakeholders

## Recommended Approach

1. Start with a clear visual design and user flow.
2. Build reusable frontend components.
3. Ensure layouts work well on small and large screens.
4. Keep content accessible and keyboard-friendly.
5. Validate interactions with simple tests.
6. Iterate quickly based on feedback.

## Notes

This document should serve as a high-level reference for building Weld and applying frontend design and app-building skills across the project.
