16 · Profile Live! 🎉
🔑 Auth methods
OAuth (Roblox, Google, Discord) → auto-fills identity, skips email confirm
Email + password → requires confirm email step before entering builder
Draft saves locally on every change; syncs to account once signed in
🎭 Dev vs Studio split
Screens above show Developer path (6 steps)
Studio swaps Role step: Rate → Team size + Hiring status + Budget
Studio Work step = Top Games (3 max) not Projects
Studio has no Portfolio Links step (5 steps total)
🔗 Social linking notes
Roblox: required for trust badge + avatar (URL fallback available)
Discord: shown prominently — studios DM devs there
GitHub, X, YouTube: optional, each boosts trust score
✅ Step map (Dev, 6 steps)
0 Type selection
1 Identity: name · Roblox · socials · colour · availability
2 Role: experience + rate type + amount
3 Bio: short + extended
4 Skills: up to 5 with descriptions
5 Work: up to 3 projects + stats
6 Portfolio: links → preview → publish
